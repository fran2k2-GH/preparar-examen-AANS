4.2. Ejemplo práctico: asignación a segmento por distancia euclídea

[Profesor - Enunciado]

- Segmento C: (2.1, 35.2, 2.3)

[Profesor - Solución]

Para determinar a qué segmento pertenece un empleado con valores (6, 37, 1), calculamos la distancia euclidiana entre estos valores y los centroides de los tres segmentos.

La distancia euclidiana entre dos puntos en un espacio tridimensional (x1, y1, z1) y (x2, y2, z2) se calcula como:

Distancia = sqrt((x2 - x1)^2 + (y2 - y1)^2 + (z2 - z1)^2)

Las distancias del empleado a los centroides son:

Segmento B: 7.07
Segmento C: 4.49

El empleado pertenece al Segmento C, ya que es el más cercano.

4.5. Ejemplo práctico: PCA y proyección sobre componentes principales

[Profesor - Enunciado]

En un análisis de Componentes Principales (PCA) en un espacio tridimensional, se han calculado las siguientes cargas para las tres primeras componentes principales:

Componente Principal 1 (CP1): (0.6, 0.5, 0.4)
Componente Principal 2 (CP2): (0.7, -0.4, 0.2)
Componente Principal 3 (CP3): (-0.3, 0.8, 0.5)

Dado un punto con valores originales (2, 4, 6), calcula su representación en el sistema de coordenadas definido por estas componentes principales.

[Profesor - Solución]

Proyección en la Componente Principal 1 (CP1):

(2, 4, 6) · (0.6, 0.5, 0.4) = (2 · 0.6) + (4 · 0.5) + (6 · 0.4) = 1.2 + 2.0 + 2.4 = 5.6

Proyección en la Componente Principal 2 (CP2):

(2, 4, 6) · (0.7, -0.4, 0.2) = (2 · 0.7) + (4 · -0.4) + (6 · 0.2) = 1.4 - 1.6 + 1.2 = 1.0

Proyección en la Componente Principal 3 (CP3):

(2, 4, 6) · (-0.3, 0.8, 0.5) = (2 · -0.3) + (4 · 0.8) + (6 · 0.5) = -0.6 + 3.2 + 3.0 = 5.6

Resultado final: el punto (2, 4, 6) en el espacio definido por las componentes principales se representa como:

(5.6, 1.0, 5.6)

4.6. Ejemplo práctico: K-Means manual con Lloyd's K-Means

[Profesor - Enunciado]

Dada la siguiente serie de 6 puntos bidimensionales Pi y unos centroides iniciales Ck, ejecuta el algoritmo K-Means (Lloyd's K-Means). Indica paso a paso cómo es la asignación y cálculo de centroides.

Puntos Pi (2D):
P1 = (2, 2)
P2 = (3, 1)
P3 = (4, 4)
P4 = (6, 5)
P5 = (8, 7)
P6 = (9, 4)

K = 2

Centroides iniciales:
C1^(0) = (3, 3)
C2^(0) = (7, 5)

Criterio: el algoritmo termina si la distancia euclidiana entre cada centroide en iteración t y t+1 es < ε = 1, o si se alcanza max_iter = 2. Contamos la primera asignación como iteración 1.

[Profesor - Solución]

Iteración 1. Asignación de clusters con distancias euclidianas.

Para cada punto calculo distancia a C1^(0) y a C2^(0) y asigno al más cercano.

Distancia euclidiana:

d((x1, y1), (x2, y2)) = sqrt((x1 - x2)^2 + (y1 - y2)^2)

Para P1 = (2, 2):

A C1^(0) = (3, 3): dx = 2 - 3 = -1 -> dx^2 = 1; dy = 2 - 3 = -1 -> dy^2 = 1.
d = sqrt(1 + 1) = sqrt(2) = 1.4142.

A C2^(0) = (7, 5): dx = 2 - 7 = -5 -> 25; dy = 2 - 5 = -3 -> 9.
d = sqrt(25 + 9) = sqrt(34) = 5.8309.

Asignación: P1 -> C1.

Para P2 = (3, 1):

A C1^(0): dx = 3 - 3 = 0 -> 0; dy = 1 - 3 = -2 -> 4.
d = sqrt(0 + 4) = 2.0000.

A C2^(0): dx = 3 - 7 = -4 -> 16; dy = 1 - 5 = -4 -> 16.
d = sqrt(16 + 16) = sqrt(32) = 5.6569.

Asignación: P2 -> C1.

Resumen asignaciones iteración 1:

Cluster 1 (cercano a C1): P1, P2, P3.
Cluster 2 (cercano a C2): P4, P5, P6.

Re-cálculo de centroides:

Nuevo C1^(1) con P1=(2,2), P2=(3,1), P3=(4,4):
x: (2 + 3 + 4) / 3 = 9 / 3 = 3.0000
y: (2 + 1 + 4) / 3 = 7 / 3 = 2.3333
C1^(1) = (3.0000, 2.3333)

Nuevo C2^(1) con P4=(6,5), P5=(8,7), P6=(9,4):
x: (6 + 8 + 9) / 3 = 23 / 3 = 7.6667
y: (5 + 7 + 4) / 3 = 16 / 3 = 5.3333
C2^(1) = (7.6667, 5.3333)

Verificación del criterio de convergencia:

Para C1:
C1^(0) = (3, 3), C1^(1) = (3.0000, 2.3333)
dx = 3.0000 - 3 = 0.0000 -> dx^2 = 0.0000
dy = 2.3333 - 3 = -0.6667 -> dy^2 ≈ 0.4445
Distancia δ1 = sqrt(0 + 0.4445) ≈ 0.6667

Para C2:
C2^(0) = (7, 5), C2^(1) = (7.6667, 5.3333)
dx = 7.6667 - 7 = 0.6667 -> dx^2 ≈ 0.4445
dy = 5.3333 - 5 = 0.3333 -> dy^2 ≈ 0.1111
Distancia δ2 = sqrt(0.4445 + 0.1111) ≈ 0.7454

Ambas distancias δ1 y δ2 son menores que ε = 1.

Por tanto, se cumple el criterio de convergencia en la iteración 1 y el algoritmo se detiene. No es necesario hacer la segunda iteración porque el desplazamiento de ambos centroides es menor que ε.
